What I've been up to.

Below is some of my recent work.
For more you can visit my GitHub.

Community Twitch Development

A web app that helps Twitch streamers collaborate with viewers on a development project.
Built with React, Redux, Node, MongoDB, Socket.io, and Passport.js.

URL Shortener

Uses Node to store user inputed links in MongoDB, returning a shortened alternative.
Front end is written in vanilla Javascript and uses HTML5 form validation.

Local Weather

A jQuery application that uses geolocation with an IP fallback,
and the darksky.net API to display your local weather.

Twitch.tv Programmers Tracker

A jQuery application that filters and displays programmers streaming at Twitch.tv/creative.
Uses Handlebars.js for templating each streamer's information.

Tic Tac Toe

Tic Tac Toe with multiple difficulty levels using the minimax algorithm.
Written in Javascript with jQuery for UI.
